# Cognifyz Software Development Internship - Python Projects

This repository contains all the tasks completed during my **Software Development Internship** at Cognifyz Technologies.

The internship provided hands-on experience in Python programming, problem-solving, file handling, CRUD operations, and web scraping. Each task was designed to strengthen fundamental and advanced software development concepts.

## 📋 Tasks Completed

### 🎮 Task 1: Text-Based Game

**Objective:** Develop a simple text-based game using conditional statements.

**Features:**

* Interactive gameplay
* User input handling
* Conditional logic
* Score/result display

### 🔢 Task 2: Number Pattern Generator

**Objective:** Generate and print number patterns using loops.

**Features:**

* Pattern generation
* Nested loops
* Dynamic user input

### ✅ Task 3: CRUD Task Management System

**Objective:** Create a console application to manage tasks.

**Features:**

* Create tasks
* View tasks
* Update tasks
* Delete tasks
* Menu-driven interface

**Objective:** Convert temperatures between Celsius and Fahrenheit.

**Features:**

* Fahrenheit to Celsius conversion

* User-friendly interface

**Objective:** Store task data permanently using file handling.

**Features:**

* Load tasks from a file

* Exception handling

### 🌐 Task 6: Web Scraping Application
